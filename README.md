# Online Retail Sales Analytics System

## Overview

Online Retail Sales Analytics System is a SQL-based database project designed to manage customers, products, orders, payments, employees, and delivery operations in an online retail business.

The project demonstrates database design, table relationships, primary and foreign keys, and business-oriented data management using SQL.

## Features

* Customer Management
* Product Management
* Order Processing
* Payment Tracking
* Employee Management
* Delivery Tracking
* Retail Sales Data Organization

## Database Schema

The database contains the following tables:

1. user_login
2. customers
3. employment_type
4. employees
5. payment
6. orders
7. product_items
8. order_items
9. order_delivery
